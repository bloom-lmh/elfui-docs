# 路由视图

`<elf-router-view>` 渲染当前匹配 record 的页面组件。

```html
<elf-router-view></elf-router-view>
```

## 嵌套与命名出口

`depth` 选择嵌套层级，`name` 选择 record 的命名视图：

```html
<elf-router-view depth="1"></elf-router-view> <elf-router-view name="aside"></elf-router-view>
```

## Route props 与自定义视图

页面通过 record 的 `props` 接收 route params 或计算结果。RouterView 也能通过默认作用域插槽拿到 `Component`、`route`、`record`、`props`、`depth` 和 `name`，适合包裹 layout、错误展示或自定义过渡。

## 异步页面与过渡

```ts
{
  path: "/reports",
  component: () => import("./pages/Reports")
}
```

异步模块可以默认导出组件，或只导出一个可识别的组件。`<elf-router-view transition="fade" duration="200">` 可在页面替换时附加 enter/leave class。
