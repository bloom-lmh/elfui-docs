# 导航

模板里使用 `<elf-link>`；它是 `<elf-router-link>` 的兼容别名。

```html
<elf-link to="/users/1">用户</elf-link>
<elf-link :to=${{ name: "user", params: { id: 1 } }}>个人页</elf-link>
```

它支持 `replace`、`active-class`、`exact-active-class`。按住 Ctrl/Cmd/Shift 点击会保留浏览器默认行为。

## 编程式导航

```ts
const router = useRouter();

void router.push({ name: "user", params: { id: 1 } });
void router.replace("/");
router.back();
router.forward();
router.go(-2);

const target = router.resolve("/users/1");
```

`push()` 和 `replace()` 在守卫取消、重定向或重复导航时返回 `NavigationFailure`；使用 `isNavigationFailure()` 判断。

## 当前路由

```ts
const route = useRoute();
const id = useComputed(() => route.params.id);
```

`route` 包含 `path`、`fullPath`、`params`、`query`、`hash`、`meta` 和完整 `matched` 链。

## 自定义 link

`useLink({ to })` 返回 `href`、`isActive`、`isExactActive` 和 `navigate()`。也可为 `<elf-link custom>` 提供默认作用域插槽，自行决定最终渲染元素。
