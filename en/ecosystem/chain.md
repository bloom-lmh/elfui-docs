# Chain

`@elfui/chain` is the optional chain-style API for projects that prefer builder composition. It belongs to the ecosystem rather than the macro-component path, so the core component documentation stays focused.

Use Chain when its fluent assembly model fits your project. Use macro components when you want TypeScript file components and compile-time templates.

See the [Chain API](/en/api/chain) for the available runtime surface.
