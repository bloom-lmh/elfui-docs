# 交互控制

交互控制函数用于弹层、抽屉、提示和可访问性场景。

## ESC 关闭

```ts
useEscapeKey(() => {
  open.set(false);
});
```

## 滚动锁定

```ts
useScrollLock(() => open.value);
```

当 `open` 为 true 时锁定 `document.body` 滚动，组件卸载时会自动恢复。

## 焦点陷阱

```ts
const host = useHost();

useFocusTrap(host);
```

::: warning
适合 Dialog、Drawer、PopConfirm 这类需要把 Tab 焦点限制在组件内部的场景。
:::
