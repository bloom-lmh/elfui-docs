# 路由视图

`<elf-router-view>` 渲染当前匹配的页面组件。

```html
<elf-router-view></elf-router-view>
<elf-router-view depth="1"></elf-router-view>
<elf-router-view name="aside"></elf-router-view>
```

`depth` 选择嵌套路由层级，`name` 选择命名视图。页面 props 可来自 params、静态对象或 route-to-props 函数；命名视图可使用以视图名为键的 props map。

默认作用域插槽可得到 `Component`、`route`、`record`、`props`、`depth` 和 `name`，适合封装页面壳、错误展示或自定义过渡。

异步页面模块会跨 RouterView 缓存，可默认导出组件或只导出一个可识别组件。`transition="fade" duration="200"` 会在页面替换时附加 enter/leave class。
