# 路由守卫

守卫可以返回 `false` 取消导航，返回路径或命名 location 重定向，也可以返回 Promise。

```ts
router.beforeEach((to) => {
  if (to.meta.requiresAuth && !auth.loggedIn.value) return "/login";
});

router.beforeResolve((to) => {
  return preloadPageData(to);
});

router.afterEach((to, from) => {
  analytics.page(to.fullPath, from.fullPath);
});

router.onError((error) => reportError(error));
```

## 路由级守卫

```ts
{
  path: "/admin",
  component: AdminPage,
  beforeEnter: () => canEnterAdmin() || "/login"
}
```

## 组件内守卫

```ts
onBeforeRouteLeave(() => !dirty.value || window.confirm("放弃修改？"));

onBeforeRouteUpdate((to) => {
  loadUser(to.params.id);
});
```

当前 Router 提供 leave/update 两种组件内守卫；没有 Vue Router 的 `beforeRouteEnter` 等实例回调语义。
