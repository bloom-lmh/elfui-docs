# 路由快速开始

路由独立于 `@elfui/core`，已有应用需要单独安装：

```bash
pnpm add @elfui/router
```

新项目也可以在脚手架交互中选择 Router，或直接执行：

```bash
pnpm create elfui@beta my-app --router --install
```

脚手架会自动安装路由包并生成 `src/router/index.ts`。

创建路由：

```ts
import { createRouter } from "@elfui/router";

export const router = createRouter({
  mode: "hash",
  routes: [
    { path: "/", component: () => import("./pages/Home") },
    { path: "/users/:id", component: () => import("./pages/User") }
  ]
});
```

页面中使用：

```html
<elf-link to="/">首页</elf-link> <elf-router-view></elf-router-view>
```

`createRouter()` 会设置 active router 并注册路由元素。

在应用入口先导入 router 模块，再挂载根组件：

```ts
// main.ts
import "./router";
import { createApp } from "@elfui/core";
import App from "./App";

createApp(App).mount("#app");
```
