# 监听器

`watch()` 用来监听明确的数据源。

```ts{1}
watch(search, (value, oldValue) => {
  console.log(value, oldValue);
});
```

数据源可以是 ref、getter 或数组：

```ts{1}
watch(
  () => user.name,
  (name) => {
    console.log(name);
  }
);
```

## watchEffect

`watchEffect()` 会自动追踪函数中读取的响应式依赖：

```ts{1}
watchEffect(() => {
  console.log(user.name, count.value);
});
```

`onWatcherCleanup()` 可在 `watch()` 或 `watchEffect()` 的同步回调内注册同一轮的清理任务：

```ts{1}
watchEffect(() => {
  const timer = window.setTimeout(refresh, 300);
  onWatcherCleanup(() => window.clearTimeout(timer));
});
```

## flush 模式

```ts{1}
watch(source, callback, { flush: "post" });
```

| 模式   | 含义           |
| ------ | -------------- |
| `pre`  | 默认调度队列   |
| `post` | DOM 更新后队列 |
| `sync` | 同步执行       |

需要清理异步任务时，在回调里使用 `onCleanup`。
