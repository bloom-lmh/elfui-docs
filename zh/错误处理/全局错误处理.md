# 全局错误处理

错误和警告配置在 `createApp()` 返回的实例上，并且应在 `mount()` 前完成：

```ts
import { createApp } from "@elfui/core";

const app = createApp(App);

app.config.errorHandler = (error, info) => {
  reportError(error, { info });
};

app.config.warnHandler = (message, ...args) => {
  console.warn("[ElfUI]", message, ...args);
};

app.mount("#app");
```

`errorHandler` 接收组件 setup/render 错误；`warnHandler` 接收该 App 组件上下文中的框架警告，例如 expose 覆盖 host 原有属性。

::: warning
不要把业务恢复逻辑放进全局 handler。可局部恢复的错误应由 `onErrorCaptured()` 或 `errorBoundary()` 处理。
:::
