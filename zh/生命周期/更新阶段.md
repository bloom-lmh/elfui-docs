# 更新阶段

更新阶段用于观察组件内部响应式绑定触发的 DOM 更新。

```ts{1-3}
onBeforeUpdate(() => {
  console.log("before update");
});

onUpdated(() => {
  console.log("updated");
});
```

## 使用建议

::: warning
优先通过响应式表达式驱动 UI，不要把普通状态同步都放进更新 hook。
:::

::: tip
适合放在更新 hook 的事情：
:::

- 读取更新后的布局尺寸
- 调整滚动位置
- 与第三方 DOM 插件同步

如果只是监听某个状态变化，用 `watch()` 更直接。
