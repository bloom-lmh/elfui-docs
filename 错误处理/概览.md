# 错误处理概览

ElfUI 的错误处理分四层：

| 层级     | API                                                   |
| -------- | ----------------------------------------------------- |
| 组件捕获 | `onErrorCaptured`                                     |
| 错误边界 | `errorBoundary` / `captureError`                      |
| 全局处理 | `createApp(App).config.{ errorHandler, warnHandler }` |
| 编译诊断 | Vite 插件和宏编译器诊断                               |

业务组件优先在局部处理可恢复错误。全局 handler 适合日志上报和兜底提示。
